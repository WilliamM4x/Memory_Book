Este é um pacote que utiliza Regex para verificar a força de uma senha segundo alguns críterios: 
Inválida: tamanho menor que 8 caracteres;
Fraca: Só letras (maiúsculas ou minúculas), números ou caracteres especiais;
Moderada: Dentre as possibilidade(letras, números, especiais)) deve haver ao menos 3;
Forte: Contém ao menos uma letra maiúscula, minúscula, um número e um caracter especial.
